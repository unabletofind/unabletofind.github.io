Sigma Rule Example — Suspicious WMI Execution 

title: Suspicious WMI Process Creation
id: 12345678-1234-1234-1234-123456789012
status: experimental
description: Detects suspicious WMI process creation often used for lateral movement
author: Swetha Devi
date: 2024/01/16
logsource:
    category: process_creation
    product: windows
detection:
    selection:
        ParentImage|endswith: '\WmiPrvSE.exe'
        Image|endswith: '\cmd.exe'
    condition: selection
falsepositives:
    - Legitimate administrative scripts
level: high
tags:
    - attack.lateral_movement
    - attack.t1047

Sample Detection Query


-- Suspicious PowerShell Execution
SELECT 
    process_name,
    command_line,
    user_name,
    COUNT(*) as count
FROM 
    process_events
WHERE 
    process_name LIKE '%powershell%'
    AND (command_line LIKE '%hidden%' 
         OR command_line LIKE '%encodedcommand%'
         OR command_line LIKE '%downloadstring%')
GROUP BY 
    process_name, command_line, user_name
HAVING 
    COUNT(*) > 3
ORDER BY 
    count DESC;
